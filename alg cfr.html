Algoritmi cu cifrele unui număr

Acest tip de algoritmi lucrează cu următoarele expresii:
n%10=ultima cifră a lui n
n/10= numărul n din care s-a eliminat ultimă cifră
Algoritmii uzuali sunt:
- aflarea sumei cifrelor unui număr
- aflarea numărului de cifre pe care le are un număr
- aflarea oglinditului unui număr
- suma cifrelor unui număr dacă n=732848, atunci suma cifrelor sale este 7+3+2+8+4+8=32 Pentru a afla suma cifrelor parcurgem pașii:
aflăm ultima cifră din n o adunăm la o variabilă s inițializată cu 0, eliminăm cifra adunată din n repetăm pașii anteriori pȃnă cȃnd n=0 ȋn pseudocod, avem următorul algoritm:
citește n
s=0
cȃt timp n?0 execută
c=n%10
s=s+c
n=n/10
scrie s
- numărul cifrelor unui număr: dacă n=732848, atunci numărul de cifre este 6 Pentru a afla numărul de cifre parcurgem pașii: crește cu 1 o variabilă contor nr inițializată cu 0 eliminăm cifra adunată din n repetăm pașii anteriori pȃnă cȃnd n=0 ȋn pseudocod, avem următorul algoritm:
citește n
nr=0
cȃt timp n?0 execută
nr=nr+1
n=n/10

scrie nr

- aflarea oglinditului unui număr: dacă n=732848, atunci oglinditul său este 848237 Pentru a afla oglinditul parcurgem pașii: aflăm ultima cifră din n o adăugăm la sfȃrșitul unei variabile inv inițializată cu 0, eliminăm cifra adunată din n repetăm pașii anteriori pȃnă cȃnd n=0 pentru exemplul dat avem următoarele operații: n=732848 deci c=8 ; inv=8; n=73284
n=73284 deci c=4; inv=84; n=7328
n=7328 deci c=8; inv=848; n=732 //cum ajung de la 84 la 848 ȋn calculul lui inv? ce operație se folosește?
n=732 deci c=2; inv=8482; n=73
n=73 deci c=3; inv=84823; n=7
n=7 deci c=7; inv=848237; n=0
n=0 => ne oprim din structura repetitivă.
Ca să adaugăm la sfȃrșitul lui inv o cifră se folosește operația inv=inv*10+n%10

Ȋn pseudocod, avem următorul algoritm:
citește n
inv=0
cȃt timp n=0 execută
c=n%10
inv=inv*10+c //aici se adaugă c la sfȃrșitul lui inv.
n=n/10

scrie inv

Observație: un număr este palindrom dacă este egal cu oglinditul sau (exemplu: 11, 121, 3, 36763, etc.)
