Programs and Features. Indexing options. Default programs
(Windows 7) (ȋn romȃnă)

Programs and Features
- Din „Progams and Features” puteți dezinstala din computer un program. Pe lȃngă aceasta puteți schimba, vizualiza și repara programe.
- Această opțiune poate fi dezactivată din „Windows Features”.

Dezinstalarea unui program: Start -> Control Panel -> Programs -> Programs and Features -> Click pe programul pe care doriți să-l dezinstalați -> Uninstall
OBS.: Opțiunile pot fi Uninstall, Change, Uninstall/Change, Rapair sau Change/Repair

Schimbarea unui program:
- ajută la instalarea/dezinstalarea anumitor caracteristici ale unui program fără a-l dezinstala.
- Nu toate programele au aceasta caracteristică.
Start -> Control Panel -> Programs -> Programs and Features -> Click pe programul pe care doriți să-l dezinstalați -> Change

Repararea unui program:
- pentru fișierele corupte și setările programului care au fost ȋnlocuite cu cele originale din dosarul de instalare.
- Cȃteodată programul este doar reinstalat cu fișierele componente suprascrise.
- Nu toate programele au aceasta opțiune.
Start-> Control Panel -> Programs -> Programs and Features -> Click pe programul pe care doriți să-l dezinstalați -> Repair

Indexing options
- Face parte din Windows 7
- Cu Windows 7 puteți face ca, căutătorul de indexuri să vă indexeze fișierele dorite, ci și conținutul acestora.

Start -> scrieți ȋn cutia de căutare ”Search”-> Indexing Options

- ȋn partea de sus veți vedea numărul de obiect indexate, apoi mai jos o listă cu locațiile care au fost incluse pe lista indexării.
- Ca să adăugați o nouă locație de indexare, click pe „Modify”. Va apărea orice drive este disponibil. Puteți bifa orice drive și dosar doriți să fie inclus pe index. Căutătorului de indexuri i-ar putea lua ceva timp să adauge tot ce doriți să fie indexat. Dacă ați observat că nu au fost indexate anumite fișiere din listă, chiar dacă le-ați selectat din listă, atunci trebuie să adăugați tipul fișierului (extensia).

- ȋn „Index Setting” puteți alege să indexați și fișiere criptate sau să modificați diferite setări ale indexului.
- Dacă ați avut probleme cu indexul, ȋl puteți reface/reconstrui făcȃnd click pe „Rebuild”.

Default programs
- un program implicit e un program pe care Windows ȋl folosește pentru a deschide un tip particular de fișier (de exemplu muzică, imagine, pagină web)(De exemplu: Dacă aveți instalat mai mult de un browser pe calculator, puteți alege care dintre ele să fie cel implicit pentru deschiderea paginilor web. )

Deschiderea „Default Programs”: Start->Default Programs
Aici puteți:
1)să alegeți programele implicite
2)să asociați un tip de fișier unui program
3) să schimbați setările redării automate
4) să stabiliți programelor implicite pentru calculator

1)Alegerea programului implicit:
- această funcție este folosită pentru alegerea programelor pe care doriți să le folosiți in Windows ȋn mod implicit.

2)Asocierea unui tip de fișiere unui program:
- această funcție este folosită pentru alegerea unui program implicit pe baza tipului de fișier sau de protocol. De exemplu: fișierele de imagine cu extensia „.jpg” pot fi deschise cu un program, iar cele cu extenia „.bmp” cu un alt program.
Start -> Default Programs -> „Asociate a file type or protocol with a program” -> Click pe tipul de fișier/protocol pe care doriți ca un anumit program să-l deschidă ȋn mod implicit -> Change Program -> Alegeți programul pe care doriți să-l folosiți pentru deschiderea fișierului respectiv sau pe săgeata de la „Other Programs” sau „Browse” pentru a găsi programul dorit -> OK

3)Modificarea setărilor de redare automată:
- această funcție este folosită pentru schimbarea programului pe care doriți să-l utilizați pentru a porni diverse drivere, adică CD-uri, DVD-uri, USB-uri etc.

4)Setarea programelor implicite:
- această funcție este folosită pentru navigarea pe internet și trimiterea de e-mail-uri.
